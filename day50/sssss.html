css选择器：

　　1 基本选择器：

　　　　元素选择器：{ css样式}

 span{color: red；}  标签选择器
.c1{color: red；}     类选择器
#d1{color: red；}   id选择器

　　2 通用选择器：通常用来指定全局的样式

*{

    font-size: 14px;
    margin: 0;
    padding: 0;
}

　　3 组合选择器

后代选择器(div p) div标签内所有的p标签
子选择器(div>p) div标签中的下一级的p标签（只找div的子标签）
毗邻选择器(div+p) div标签后第一个p标签，div和p是同级
弟弟选择器(div~p) div标签后的所有p标签，div和p是同级

 

 

 

 

　　　　组合选择器不但可以只用标签选择器，也可以通过属性加标签和属性加属性的方式。

　　　　例如：

 　　　　　　#d1 .c1    意思就是id=d1 的标签包含的所有class=c1标签

　　　　　　 div .c1     意思就是div标签包含的素有class=c1标签

　　4 属性选择器：

[s10] 属性为s10
[s10='bbbb'] 属性s10='bbbb'
[s10^='bbbb'] 属性s10值以bbbb开头
[s10$='bbbb'] 属性s10值以bbbb结尾
[s10*='bbbb'] 属性s10值中包含bbbb
[s10~='bbbb'] 属性s10值中以空格隔开的含有bbbb的

 

 

 

 

 

　　5 分组选择器和嵌套选择器：

　　　　分组：

div, p {
  color: red;
}

　　　　嵌套：

.c3 div {
  color: green;
}

class=c3的标签下的div标签

　　6 伪类选择器和伪元素选择器：

　　　　伪类选择器：

link 未访问的链接 a:link{color:red;} 未点击的链接显示样式
visited 已访问的链接 a:visited{color:red;} 以点击的链接显示样式
hover 鼠标移动到链接上 a:hover{color:red;} 鼠标移动到链接显示样式
active 选定的链接 a:active{color:red;} 鼠标左键点击链接显示样式
focus input输入框获取焦点时的样式 input:foucs{outline:none;backgroud-color:#eee} 鼠标点击input输入框时的显示样式

 

 

 

 

 

　　　　伪元素选择器：

first-letter h1:first-letter{font-size:48px;} h1标签内容第一个文字大小为48px
before a:before{content:"#"} a标签之前加上#
after div:after{content:"" display:block;} div标签之后加上空内容,且为块级元素

　　5 display 控制标签的显示效果

none 在浏览器中不显示

inline 以行级元素进行展示
inline-block 同时有块级和行级元素的特性

 

 

 

 

 　　　visibility:hidden 可以隐藏某个元素，但是该元素的空间无法隐藏。

　　　 display:none可以隐藏元素，同时隐藏该元素的空间

　　

 

　　6 浮动

　　　　css中任何元素都可以浮动，浮动后回生成一个块级元素。可以向做或向右移动，直到他的外边缘碰到别的框边。浮动框已经脱离了文档的普通流中，就像ps的图层一样。这样就可能会造成父标签塌陷无法展示（恰巧父标签没有指定行高或没有其他的内容或元素）

float:none 默认。没有浮动
float:right 向右浮动
float:left 向左浮动

 

 

 

    <style>
        .c1 {
            width: 20%;
            height: 500px;
            background-color: darkturquoise;
            float: left;
        }
        .c2 {
            width:80%;
            height: 500px;
            background-color: red;
            float: left;

        }
    </style>

　　

　　7 清除浮动clear

　　　　clear 只能清除自己标签的浮动，并不能对浮动的标签起作用

clear:left 清除左侧浮动
clear:right 清除右侧浮动
clear:both 清除左右两侧浮动
clear:none 允许浮动元素出现在两侧，默认值
clear:inherit 从父元素继承clear属性值

 

 

 

 

 　　　　伪元素清除浮动：

        #d1:after{
            content: "";
            clear: both;
            display: block;
        }

　　　　同时还有 固定高度和 overflow：hidden清除浮动

 

　　8 溢出属性overflow：

　　　　当标签内内容多的使标签完全包裹的时候，就会溢出。比如在使用博客园插入表格的时候，表格超过2个化，会造成表格错乱。就可以使用overflow来清除。

overflow-x 水平方向
overflow-y 垂直方向
overflow:visible 默认。如果不能包裹就溢出
overflow:hidden 超过边框部分隐藏
overflow:scroll 如果内容溢出，就出现滚动条查看
overflow:auto 如果内容溢出，就出现滚动条查看
overflow:inherit 从父元素继承overflow属性值

 

 

 

 

 

 

　　9 定位 position

position:static 默认无定位
position:relative 相对定位。相对标签以前的位置进行定位
position:absolute 绝对定位。相对已经定位的父标签进行定位。若没有就会一直找到body标签
position:fixed 固定定位

 

 

 

 

 

　　10 z-index 对象层叠顺序

　　　　数值越大越靠近你。只有定位的元素才能使用z-index，浮动的不行

.c1{
z-index:1000
}

　　       经典应用模态框

 

 

　　11 opacity透明度：

　　　　取值0-1。opacity改变元素，子元素的透明效果，rgba只改变背景颜色透明度
